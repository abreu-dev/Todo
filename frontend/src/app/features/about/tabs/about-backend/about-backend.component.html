Sobre o desenvolvimento

do Backend

O backend do projeto foi todo feito em C#, com .NET 5.0. A estrutura é em camadas com uma API REST.

O gerenciamento de contas havia sido desenvolvido um modelo próprio, porém, ao chegar em questões sobre segurança de dados foi alterado para utilizar o Microsoft Identity.
Os princípios do TDD foram seguidos, contando com testes de unidade e de integração. Outras metodologias foram implementadas também, como Repository Pattern, CQRS e IoC.
