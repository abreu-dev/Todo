Sobre o desenvolvimento

do frontend

O frontend do projeto foi todo feito em Angular, versão 11.0.7 e Node 15.6.0. Foi utilizado Angular Material para criar a maior parte das telas.
A estrutura foi dividida entre três principais pastas, core, features e shared. Algumas das implementações são interceptors, services, singleton services, directives e guards. Também foi configurado eslint com algumas regras recomendadas.

Inicialmente foi desenvolvimento sem pensar muito em usabilidade e estilização, mais para por em prática as principais funcionalidades
Depois disso pronto, foram realizados alterações e ajustes para melhorar a interface gráfica.
